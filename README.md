## project scope 

Bluetooth-based home automation is a simple and low-cost embedded system used for controlling home appliances using a smartphone.

## Components Used

Main Components:

- 0Arduino Uno / ESP32 (Microcontroller)
- HC-05 / HC-06 Bluetooth Module
- Relay Module (1-channel or multi-channel)
- Breadboard
- Jumper wires
  Power Supply:

- 5V USB supply or adapter
  
Load Devices:

LED bulb / AC bulb (for demo)
Fan (optional demonstration load)

Control Device:

Android smartphone with Bluetooth terminal/control app
